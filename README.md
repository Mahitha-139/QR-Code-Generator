# 📱 Simple QR Code Generator

A lightweight, beginner-friendly Python script that converts any text or website URL into a scannable QR Code image.

---

## 🚀 Features
* **Simple Configuration**: Easily change the target URL or text.
* **Custom Styling**: Adjust pixel size, borders, and foreground/background colors.
* **Instant Export**: Saves the generated QR code directly as a `.png` image file.

---

## 🛠️ Prerequisites

Before running the script, you need to have **Python 3** installed on your system. You will also need to install the `qrcode` library and the `pillow` image processing library.

Install the required dependencies using pip:
```bash
pip install qrcode pillow
```

## 💻 How to Use

1. **Clone or Download** this repository to your local machine.
2. Open `qr_generator.py` in your favorite code editor.
3. Locate the `data` variable and replace the placeholder text with your desired URL or message:
   ```python
   data = "https://your-website.com"
   ```
4. Run the script from your terminal:
   ```bash
   python qr_generator.py
   ```
5. Check your project folder for a new file named `my_qrcode.png`. Scan it with your smartphone camera!

---

## ⚙️ Customization

You can tweak the QR code settings inside the script:
* **`box_size`**: Change the value (e.g., to `15` or `20`) to make the final image larger.
* **`fill_color` & `back_color`**: Change `"black"` and `"white"` to colors like `"navy"` or `"lightgray"` to match your personal brand.
